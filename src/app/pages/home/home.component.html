Bienvenido a Mi Proyecto de Notas

Este proyecto es una demostración de mis habilidades como desarrollador frontend utilizando Angular, consumo de APIs externas y diseño de interfaces modernas.

Características del Proyecto

  • Lista de notas dinámica utilizando datos de una API externa.
  • Validación de formularios e interacción en tiempo real.
  • Estilo minimalista inspirado en diseño moderno.
  • Desarrollado con TypeScript y arquitectura modular.
